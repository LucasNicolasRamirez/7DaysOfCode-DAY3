C#(pronunciado "C-Sharp") es un lenguaje de programación orientado a objetos y parte de la plataforma .NET de Microsoft. Fue diseñado para ser sencillo, moderno yseguro , con una sintaxis que recuerda a otros lenguajes como C++ yJava. C# es utilizado principalmente para desarrollar aplicaciones de escritorio, juegos (con Unity), aplicaciones móviles (con Xamarin), y aplicaciones web (con ASP.NET).

Java es un lenguaje de programación orientado a objetos, diseñado para ser independiente de la plataforma, lo que significa que los programas escritos en Java pueden ejecutarse en cualquier sistema operativo que tenga la Máquina Virtual de Java (JVM). Java es utilizado principalmente para el desarrollo de aplicaciones empresariales, aplicaciones móviles (particularmente Android), sistemas embebidos, y aplicaciones web.
